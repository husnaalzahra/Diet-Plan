# Personalized Nutrition Planner

This project is a simple Python-based application that helps in planning a personalized diet based on user inputs. It is designed as part of a learning project to understand how basic programming can be used in real-life applications like nutrition planning.

## About the Project
The program takes user details such as age, weight, height, and lifestyle, and provides a basic nutrition plan. It shows how logic and calculations can be applied to create useful outputs.

## Features
- Takes user input for personal details  
- Calculates basic nutritional needs  
- Provides a simple diet suggestion  
- Easy to understand and use  

## Technologies Used
- Python
